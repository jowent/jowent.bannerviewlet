jowent.bannerview Installation
------------------------------

To install jowent.bannerview using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``jowent.bannerview`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        jowent.bannerview
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
