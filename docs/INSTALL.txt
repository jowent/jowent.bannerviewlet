jowent.bannerviewlet Installation
------------------------------

To install jowent.bannerviewlet using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``jowent.bannerviewlet`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        jowent.bannerviewlet
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
